轻阅读
一个支持多端同步的阅读应用后端。

项目简介
轻阅读是一个适配手机、平板等设备的阅读应用，核心特点是多端同步阅读进度。本仓库是它的服务端代码，你可以自己部署，用于同步你自己的阅读数据。

相比于其他复杂的阅读方案，它更轻量，更易于在不同设备上获得一致的体验。



核心特点
多端同步：在手机、平板、电脑上同步你的书架和阅读进度。

适配性好：针对不同屏幕尺寸的设备做了界面适配。

自建服务：你可以完全掌控自己的阅读数据。
